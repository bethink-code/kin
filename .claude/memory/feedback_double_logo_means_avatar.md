When Garth says "double logo" or "extra avatar" near the top bar / canvas pill area, he is referring to the **user avatar appearing twice** (UserMenu + the same photo embedded inside another component). Not the Ally wordmark.

**Why:** The Ally wordmark on the right of the TopBar is intentional product branding. The per-pane Ally header on the right pane is contextual ("Ally / in chat"). Both serve different purposes — wordmark = product, pane header = current mode.

I previously misread "double logo" as meaning the Ally wordmark was redundant with the per-pane Ally header, and removed the wordmark. Garth flagged this immediately.

**How to apply:**
- "Double logo" / "extra avatar" near the user avatar area = remove the duplicate avatar from whichever component embedded it (e.g. avatar inside the canvas pill).
- Don't touch the Ally wordmark unless Garth explicitly says so.
- When in doubt, ask which one before deleting.
